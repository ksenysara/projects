Web определения

Internet
Интернет - всемирная система объединённых компьютерных сетей. Интернет образует глобальное информационное пространство, служит физической основой для Всемирной паутины (WWW) и множества других систем (протоколов) передачи данных. В настоящее время под словом «Интернет» чаще всего имеется в виду Всемирная паутина и доступная в ней информация, а не физическая сеть.
WWW
World Wide Web Всемирная Паутина - основная служба в сети Интернет, позволяющая получать доступ к информации на любых серверах, подключенных к сети.
W3C
World Wide Web Consortium Консорциум Всемирной паутины - организация, разрабатывающая и внедряющая технологические стандарты для Всемирной паутины.
Browser
Браузер - программное обеспечение для просмотра веб-сайтов. Браузеры постоянно развивались со времени зарождения Всемирной паутины и с её ростом становились всё более востребованными программами.
HTML
HyperText Markup Language язык разметки гипертекста - стандартный язык разметки документов во Всемирной паутине. Большинство веб-страниц создаются при помощи языка HTML.
CSS
Cascading Style Sheets каскадные таблицы стилей - язык оформления внешнего вида веб-страниц, написанных с помощью языков разметки HTML.
